# IziToolkit
A minimalistic bash toolkit set

# Instalation
Clone project

```bash
git clone ... ~/.izi-toolkit
```

And load all or separate scripts in your bash (just add it to your .zshrc)

```bash
[ -f "$HOME/.izi-toolkit/all.zsh" ] && source "$HOME/.izi-toolkit/all.zsh"
```
